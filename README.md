Importing repository into Rstudio:
- Make sure you have Git, R and Rstudio installed
- Open Rstudio
- Go to: File --> new project --> Version Control --> Git -->
- You'll be asked for an URL, go to this github site, and it should be in the Code button, copy it and paste it on Rstudio
- Choose the folder you want to have your clone of the repository

- That should be it, if any doubts contact me
